Outbreak¶

dtk.interventions.outbreak.recurring_outbreak(cb, outbreak_fraction=0.01, repetitions=-1, tsteps_btwn=365, target='Everyone', start_day=0, strain=(0, 0))[source]¶

Function to add recurring introduction of new infections to the current configuration builder

Parameters:
  • cb – Configuration builder holding the interventions
  • outbreak_fraction – Fraction of people getting infected by the outbreak
  • repetitions – Number of repetitions
  • tsteps_btwn – Timesteps between repetitions
  • target – Target demographic. Default is ‘Everyone’
  • start_day – Start day for the outbreak
  • strain – Needs to be defined as (Antigen,Genome)
Returns:

A dictionary holding the fraction and the timesteps between events
